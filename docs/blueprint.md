# **App Name**: Gestiónia

## Core Features:

- Panel de Control Interactivo: Una vista panorámica en tiempo real de los indicadores clave de negocio, como ventas, inventario y rendimiento general.
- Gestión de Productos y Catálogo: Capacidad para añadir, editar, organizar y rastrear todos los productos, incluyendo detalles, precios y stock.
- Soporte para Venta por Peso (Kg): Permite definir y gestionar productos con unidades de venta basadas en peso (kilogramos) en el catálogo y durante el procesamiento de ventas.
- Procesamiento de Ventas y Pedidos: Herramientas para registrar nuevas ventas, gestionar el estado de los pedidos y generar facturas de forma eficiente.
- Generación y Compartir Comprobantes: Funcionalidad para generar comprobantes de venta y pago en formato PNG, con la opción de compartirlos digitalmente o imprimirlos directamente.
- Gestión de Clientes y Contactos: Un sistema para almacenar y organizar la información de clientes y proveedores, facilitando la comunicación y el seguimiento.
- Estados de Cuenta de Clientes Detallados: Proporciona un estado de cuenta pormenorizado para cada cliente, mostrando sus datos, un desglose de todas sus ventas organizadas por día y un resumen de saldos.
- Historial de Transacciones Detallado: Un registro completo de todas las transacciones financieras (ventas, compras, pagos) para una contabilidad clara.
- Informes Personalizables y Análisis: Generación de informes configurables sobre ventas, inventario, comportamiento del cliente y rentabilidad para una toma de decisiones informada.
- Insights de Negocio con IA: Una herramienta de IA que analiza los datos de ventas, productos y clientes para proporcionar predicciones y recomendaciones para optimizar estrategias y detectar tendencias.
- Configuración de Moneda Regional: Facilita la configuración de la moneda predeterminada de la aplicación, incluyendo soporte para monedas de Latinoamérica, con el Sol peruano como opción por defecto.

## Style Guidelines:

- Esquema de color general claro para profesionalismo y claridad. El color primario es un verde azulado oscuro (#3D8F72), simbolizando crecimiento y eficiencia. El color de fondo es un blanco casi verdoso muy suave (#EFF8F5) para una sensación de ligereza. El color de acento es un cian-azul vibrante (#61BBDB), utilizado para botones importantes y elementos interactivos para atraer la atención del usuario.
- La fuente 'Inter' (sans-serif) se utiliza tanto para titulares como para el cuerpo del texto. Su diseño moderno y altamente legible garantiza la claridad y la eficiencia en la presentación de datos, ideal para un entorno empresarial.
- Se utilizarán iconos de estilo moderno, minimalista y geométrico para la navegación y las representaciones de funciones, asegurando que sean fácilmente reconocibles y que mejoren la experiencia del usuario sin sobrecargar visualmente el panel.
- Diseño de panel modular y limpio con una navegación lateral intuitiva y una disposición de contenido que prioriza la legibilidad y el acceso rápido a la información. El enfoque es un equilibrio entre densidad de datos y espacios en blanco para evitar la fatiga visual.
- Se implementará un logo animado sutil en la carga de la aplicación para añadir un toque dinámico. Se incluirán microinteracciones con animaciones discretas en los elementos de la interfaz, como clics en botones o actualizaciones de datos, para proporcionar retroalimentación y una experiencia fluida al usuario.
